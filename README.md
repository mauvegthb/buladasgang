## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/mauvegthb/buladasgang.git
   ```
2. Navigate to the project directory:
   ```bash
   cd buladasgang
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the project:
   ```bash
   npm start
   ```

## Customization Guide
- To change the color scheme, update the CSS variables in `styles.css`.
- To add new pages, create a new HTML file in the root directory and link it in the navigation menu.
- Ensure you update the `index.html` file to include any new scripts or stylesheets.
